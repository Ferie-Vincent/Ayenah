# Tech Stack

- **PHP**: 8.2
- **Framework**: Laravel 11.x
- **Auth scaffold**: Laravel Breeze 2.x (Blade)
- **Frontend build**: Vite + Tailwind CSS + PostCSS
- **DB**: MySQL 8 via MAMP (port 8889)
- **Image processing**: intervention/image ^3.11
- **Security**: spatie/laravel-csp ^3.13 (Content Security Policy)
- **Sitemap**: spatie/laravel-sitemap ^7.3
- **Error tracking**: sentry/sentry-laravel ^4.13
- **Testing**: Pest 3.x + pest-plugin-laravel
- **Code style**: Laravel Pint
- **Dev server**: MAMP (Apache), php artisan serve also works
- **Node**: package.json present, npm used (package-lock.json)
- **Local URL**: http://127.0.0.1:8889 (MAMP) or http://localhost:8000 (artisan serve)
